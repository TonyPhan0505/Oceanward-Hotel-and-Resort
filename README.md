# Oceanward-Hotel-and-Resort
A web app to manage bookings, finance, discounts distribution, customer relationship and on-campus services for a hotel in Vung Tau, Vietnam. This web app has 3 user types: clients, admin and staff.

## Source code:
Hidden.

## Live link:
https://oceanward-hotels-and-resort.netlify.app

## Demo video:
https://youtu.be/RMNc4z3JK-o?si=9pOB2EFT6rrKNyuz

## Status: 
Fully functional and live.

## Tech-stack: 
1. React: to implement user interface.
2. MongoDB: to store text data. I use Mongoose to send and retrieve data to and from MongoDB.
3. Express.js: to implement controllers, routes, middleware.
4. Node.js
5. Redux toolkit and saga: to manage more than 100 app states.

## Features:
---
- Online payment: completed.
- Automatic refunding: completed.
- Secure passwords storage with hashing: completed.
- Self-implemented 2-step authentication: completed.
- Self-implemented 2-step password recovery: completed.
- Time conflicts detection: completed.
- Staff account creation for admins: completed.
- Profile management for client, admin and staff: completed.
- Booking management for staff and admin: completed.
- Discount codes distribution: completed.
- Staff accounts management for admins: completed.
